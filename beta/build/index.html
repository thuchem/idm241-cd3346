Triggers

Hover-on: When users hover on the "Appreciate" button

Hover-off: When users hover off the "Appreciate" button

Click: When users click on the "Appreciate" button

Rules

Hover-on: The cursor change to pointer, the background color of the button become a darker blue

Hover-off: The cursor change back to original arrow, the background color of the button back to bright blue

First Click: The background color of the button become dark grey, the thumb-up icon become smaller and move up, the "Appreciate" line under the button changes to "Appreciated"

Second Click: The background color of the button become back to bright blue, the thumb-up icon become bigger and placed at center, the "Appreciated" line under the button changes to "Appreciate"
